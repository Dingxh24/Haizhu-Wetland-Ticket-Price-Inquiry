# 湿地公园票价查询系统（Desktop Version）

## 项目简介

本项目是一个基于 Flask + pywebview 的桌面票价查询系统，实现以下功能：

* 免票条件判断
* 优惠票计算
* 节假日自动折扣（五一/国庆/春节）
* 桌面窗口化界面（无浏览器地址栏）

## 技术栈

* Python
* Flask
* pywebview
* chinese-calendar

## 运行方法

```bash
pip install -r requirements.txt
python launcher.py
```
